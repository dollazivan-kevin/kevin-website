WHY DESIGN-FIRST

细节，决定气质

随手出稿 vs 系统驱动——最终呈现的质感，不在一个档次。

随手出稿
/* hero.css */
background: #2a2a2a;
color: #333;
padding: 12px;
/* footer — 又换了一套 */
background: #1d1d1d;
color: rgb(45,45,45);
padding: 14px;
  • × 每个页面配色随机发散
  • × 字号字重靠感觉，改版重排
  • × 动效粗糙或根本没有
  • × 换工具 / 换需求，全盘重来
vs
设计系统驱动
/* token 驱动 */
--bg: #0a0b0e;
--text-1: #ebecef;
--sp-5: 24px;
/* 全页面复用同一套 */
.hero { background: var(--bg); }
.footer { background: var(--bg); }
  • ✓ 色板、字阶、间距全部命名
  • ✓ 跨页面风格高度一致
  • ✓ 动效分级，有据可依
  • ✓ 修一处 token，全站跟着变
